Privacy Policy

Last updated: November 28, 2025

1. Introduction

Korekt, Inc., a Delaware corporation ("Korekt," "we," "us," or "our"), is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our AI-powered code review platform.

By using Korekt, you agree to the collection and use of information in accordance with this policy.

2. Information We Collect

Account Information

  • Email address
  • Name
  • Password (stored as a secure hash, never in plain text)
  • Organization membership

Code and Repository Data

  • Source code diffs and file contents submitted for review
  • Commit messages and branch names
  • Repository URLs
  • Author information (names, email addresses) from commits

Analysis Data

  • AI-generated code review results (issues, praises, severity ratings)
  • Token usage and cost metrics
  • Analysis timestamps and processing times

Integration Credentials

  • Version control system tokens (GitHub, Bitbucket, Azure DevOps)
  • Issue tracking credentials (JIRA, Azure Boards)
  • These credentials are encrypted using AES-256 encryption

3. How We Use Your Information

We use the information we collect to:

  • Provide, maintain, and improve our code review services
  • Process and analyze your code submissions
  • Generate team analytics and contributor metrics
  • Track usage for billing and resource management
  • Communicate with you about your account and service updates
  • Ensure the security and integrity of our platform

4. AI Processing

We do not use your code to train AI models.

Your code is processed by Google's Gemini AI solely to generate code review feedback. This processing is done on a per-request basis, and your code is not retained by Google for model training purposes.

We have configured our AI integration with settings that prioritize your data privacy and prevent the use of your submissions for model improvement.

5. Third-Party Services

We use the following third-party services to provide our platform:

  • Google Cloud Platform (Gemini AI) - For AI-powered code analysis. Code is sent to Google's API for processing but is not used for training.
  • Version Control Providers - We connect to GitHub, Bitbucket, and Azure DevOps using credentials you provide to access your repositories (read-only).
  • Issue Tracking Systems - We connect to JIRA and Azure Boards to read ticket information and post analysis results as comments.

We do not use third-party analytics, advertising, or tracking services on our platform.

6. Data Storage and Security

Location: Your data is stored on Google Cloud Platform servers located in the United States (us-central1 region).

Security Measures:

  • All credentials are encrypted using AES-256 encryption
  • Passwords are hashed using bcrypt with appropriate work factors
  • Data is transmitted over encrypted connections (HTTPS/TLS)
  • Access to data is restricted on an organization basis
  • Sensitive information (API keys, tokens) is filtered from logs

7. Data Retention

We retain your data, including analysis logs and code review history, indefinitely to provide you with historical insights and analytics.

You may request deletion of your data at any time by contacting us at hello@korekt.ai. Upon receiving a verified deletion request, we will remove your data within 30 days.

8. Your Rights

You have the right to:

  • Access - Request a copy of the personal data we hold about you
  • Correction - Request correction of inaccurate personal data
  • Deletion - Request deletion of your personal data
  • Export - Request your data in a portable format
  • Withdraw Consent - Withdraw consent for data processing where applicable

To exercise these rights, contact us at hello@korekt.ai.

9. Children's Privacy

Korekt is not intended for use by individuals under the age of 18. We do not knowingly collect personal information from children.

10. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last updated" date.

Your continued use of Korekt after any changes indicates your acceptance of the updated policy.
